Background:

I worked as a Creative Technologist Intern at the data visualization studio Pitch Interactive during my final year of academic studies at UW Madison. My work with Pitch consisted of developing interactive visualization prototypes for a selection of neuroscience-adjacent projects that the studio was involved in.

One of the main projects I worked on at Pitch was development of an artistic visualization framework for EEG data. Pitch had developed a relationship with Capacitor--a modern dance troupe interested in utilizing light projections of EEG data visualization in an upcoming performance. The dancers underwent EEG scans at the local UCSF facility to generate a corpus of data to be used for this purpose.

Over the course of several weeks, I created an extensible Java application that could accept arbitrary time dimensional EEG input streams and output simple geometry-based visuals with GLSL shader effects.

Process:

I cleaned and normalized the raw EEG data in R, removing erroneous outlier values and standardizing the activation range across each of the electrodes used in the scan.

Once the data was clean I began building the components of the application. In order to allow for smooth, “Youtube-style” scrubbing interaction with the electrode time series, I created a custom fork of Till Nagel’s “Animated Time Range Slider for Processing” that supports nanosecond durations and non-range time increments. My contribution was PR'ed and added as a core functionality of the Time Slider.

I then built out the rest of the application which included a basic UI for mapping each electrode activation time series to a polygon with visual effects.

Next, I extended the feature set to include a demo mode which hides the UI to allow for full-screen visualization suitable for projection. Other features I also incorporated included GLSL shader based effects and alternate geometry modes--the effect shown here generates bursts of particles mapped to an average level of activation across all electrodes and guides these particles along a path defined by Perlin noise.

Impact:

By making the shaders and geometry rendering capabilities as modular functions, this application enabled other members of the studio to rapidly test new ideas for visuals with the same EEG data. This application was then used after my internship ended to iterate versions of the visual projections that were eventually used in Capacitor’s “Synaptic Motion” performance.

I was also able to make a contribution to the open source Processing community through my fork of the Time Range Slider, which was a fulfilling opportunity given Processing's pivotal role in educating creative technologists.
